# 🚀 Python & Data Structures Practice Roadmap

Welcome to my central repository for mastering programming fundamentals, foundational logic, and algorithmic thinking. This space is structured into 5 sequential phases to track progress and document hands-on scripts.

---

## 📂 Curriculum Structure

### 📦 [01-programming-basics](./01-programming-basics/)
*The foundational building blocks needed to write and understand basic scripts.*
*   💬 **Comments** – Code documentation.
*   📦 **Variables and Data Types** – Storing information.
*   🔤 **In-Depth Strings** – Text manipulation.
*   🖥️ **Print Statements** – Console outputs.

### 🔄 [02-program-flow](./02-program-flow/)
*Making programs interactive, data conversions, and logic control.*
*   📥 **Input Statements** – User interaction.
*   🔀 **Type Conversion & Values** – Casting and logical evaluations.
*   🛣️ **Control Flow** – Conditional paths and execution loops.

### 🧱 [03-structuring-code](./03-structuring-code/)
*Transitioning from linear scripts to scalable, modular design.*
*   ⚙️ **Functions** – Reusable, modular code blocks.
*   📊 **Data Structures** – Organizing data efficiently using native collections.

### 🛠️ [04-advanced-concepts](./04-advanced-concepts/)
*Handling real-world execution edge cases and working with external systems.*
*   ⚠️ **Error Handling** – Resilient applications with exception catching.
*   📂 **File I/O** – Reading from and writing to local disk space.
*   📍 **File Positioning** – Navigating file streams precisely using seek/tell.
*   📐 **Object-Oriented Programming (OOP)** – Custom structural classes and objects.

### 🎓 [05-practice-and-application](./05-practice-and-application/)
*Solidifying conceptual knowledge through deliberate practice and project buildout.*
*   📝 **Practice Exercises** – Coding tasks reinforcing structural topics.
*   🎬 **Capstone: Movie Script Generator** – Complete, functional application project.
